📁 Repositório do Blog

💡 Como configurar

  1. Crie um repositório no GitHub com o arquivo index.html do blog
  2. Conecte o repositório ao Netlify (ou Vercel)
  3. Gere o Token em GitHub → Settings → Developer Settings → Personal Access Tokens
  4. Preencha os campos acima e clique em "Salvar"
  5. Volte em "Novo Artigo" e publique — o blog atualiza em ~1 minuto
